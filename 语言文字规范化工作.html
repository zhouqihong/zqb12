一、标准和标准化

国家语委工作职责：拟定国家语言文字工作的方针、政策，制订语言文字工作中长期规划；制订汉语和少数民族语言文字规范和标准并组织协调监督检查；指导推广普通话工作和普通话师资培训工作。
教育部语信司工作职责：研究并审定语言文字标准和规范，拟订语言文字信息处理标准；指导地方文字规范化建设；承担少数民族语言文字规范化工作，指导少数民族语言文字信息处理的研究与应用。

（一）什么是标准

根据国家标准 GB/T 20000.1-2014《标准化工作指南第1部分：标准化和相关活动的通用术语》的定义，标准是指：通过标准化活动，按照规定的程序经协商一致制定，为各种活动或其结果提供规则、指南或特性，供共同使用和重复使用的文件。

（二）什么是标准化

根据同一国家标准，标准化是指：为了在既定范围内获得最佳秩序、促进共同效益，对现实问题或潜在问题确立共同使用和重复使用的条款以及编制、发布和应用文件的活动。

二、语言文字规范标准简介

（一）语言文字规范标准概述

语言文字是人类社会最重要的交际工具和信息载体。规范标准是明文规定或约定俗成的、以特定形式发布的、共同遵守的准则。古代语言文字规范标准的形式包括韵书、教科书、字书等。

（二）语言文字规范标准的种类

按标准化领域：农业标准、工业标准、信息技术标准等。

按标准化对象：产品标准、过程标准、服务标准。

按标准编制目的：基础标准、技术标准、公益标准。

按内容：文字标准（字形标准、字量标准、字序标准等）、语言标准（语音标准、语汇标准、语法标准等）。

按语种：国家通用语言文字标准、少数民族语言文字标准、外语应用标准。

语言文字规范标准的性质

  • 强制性标准（GB）：强制使用、必须执行的标准，保障人身健康和生命财产安全、国家安全、生态环境安全以及满足经济社会管理基本需要的技术要求。
  • 推荐性标准（GB/T）：推荐使用、自愿实施的标准。语言文字规范标准多为推荐性，推荐性占多数、强制性占少数。

语言文字规范标准的级别

  • 国家标准：如《标点符号用法》
  • 行业标准：如《信息处理用GB 13000.1字符集汉字字序规范》
  • 地方标准、团体标准、企业标准

80年代前，由全国人大和国务院直接颁布（如《汉字简化方案》《汉语拼音方案》）。80年代后，由国务院语言文字主管部门单独发布或国务院有关部门联合发布（如《普通话异读词审音表》）。国际标准如 ISO 10646《信息技术 通用多八位编码字符集》。

三、语言文字标准化工作回顾

（一）语言文字规范标准建设取得丰硕成果

上世纪五六十年代：汉字规范奠基

1955年 — 中国文字改革委员会、文化部联合发布《第一批异体字整理表》
1956年 — 国务院公布《汉字简化方案》；第一届全国人大第五次会议通过《关于〈汉语拼音方案〉的决议》
1958年 — 汉语拼音方案正式确立
1964年 — 文改会、文化部、教育部发布《关于简化字的联合通知》；中国文字改革委员会编印《简化字总表》
1965年 — 中国文字改革委员会、文化部联合发布《印刷通用汉字字形表》，确定了每个汉字的笔画数和笔画结构

改革开放以来：规范标准体系逐步完善

1985年 — 国家语委、国家教委、广播电视部联合发布《普通话异读词审音表》
1988年 — 国家语委、国家教委联合发布《现代汉语常用字表》（分常用字和次常用字两部分）；国家语委、新闻出版署联合发布《现代汉语通用字表》（收7000字）
1997年 — 国家语委、新闻出版署联合发布《现代汉语通用字笔顺规范》，规定了7000个汉字的笔顺
1999年 — 国家语委发布《GB13000.1字符集汉字字序规范（笔画序）》

信息领域语言文字标准建设

语言文字负载着社会信息的80%左右，要促进国家信息化，首先需要语言文字的信息化。1986年全国语言文字工作会议开始关注信息领域的语言文字标准问题；1997年全国语言文字工作会议把信息领域语言文字标准建设列为国家语言文字工作的重要内容；2020年全国语言文字会议提出要不断完善国家通用语言文字的规范标准体系。

  • GB 2312-1980《信息交换用汉字编码字符集基本集》——国家标准总局1980年发布，规定了汉字信息交换用的基本图形字符及其二进制编码表示，共收汉字6763个。
  • ISO/IEC 10646《信息技术通用多八位编码字符集》——被当今国际社会普遍接受、信息界普遍采用的多文种国际编码标准，是信息处理最基础性的核心编码标准。
  • GF 3006-2001《汉语拼音方案的通用键盘表示规范》——国家语委2001年发布。
  • GB/T 20532-2006《信息处理用现代汉语词类标记规范》——国家质检总局、国家标准委2006年发布。

新时代重要规范标准

《通用规范汉字表》（2013年6月5日国务院公布）：共收8105字，分为三级。一级字表为常用字集，收字3500个；二级字表收字3000个；三级字表收字1605个，收录姓氏人名、地名、科学技术术语、中小学语文教材文言文用字等。

《公共服务领域外文译写规范》（2011年启动研制）：涉及英、俄、日等语种，涵盖交通、旅游、文娱、体育、卫生、商贸、餐饮住宿等领域。《公共服务领域英文译写规范第1部分：通则》于2013年12月31日发布。

新时期普通话审音：全面总结国语运动一百年、特别是新中国70多年来语音规范化的成果，建立健全普通话语音规范标准体系，修订《普通话异读词审音表》。

《标点符号用法》（GB/T 15834-2011）：国家质检总局、国家标准委2011年发布，规定了现代汉语标点符号的用法，修订并代替了1995年版本。

《出版物上数字用法》（GB/T 15835-2011）：国家质检总局、国家标准委2011年发布，规定了出版物上汉字数字和阿拉伯数字的用法。

《中国人名汉语拼音字母拼写规则》：国家质检总局、国家标准委2011年发布，在中国文字改革委员会1974年发布的《中国人名汉语拼音字母拼写法》基础上制定，规定了使用汉语拼音字母拼写中国人名的规则。

国家通用手语和盲文规范：国家语委会同中国残联分别于2018年、2021年、2025年联合发布《国家通用手语常用词表》《国家通用盲文方案》《国歌国家通用手语方案》《国歌国家通用盲文方案》《国家通用手语水平等级标准及测试大纲》等，保障听力残疾人和视力残疾人使用手语和盲文的语言权利。

《中国英语能力等级量表》：国家语委2018年4月发布，2025年1月修订。首个面向我国英语学习者的英语能力测评标准，将学习者英语能力从低到高划分为基础、提高和熟练三个阶段，共设九个等级。

《国际中文教育中文水平等级标准》：国家语委首个面向外国中文学习者的规范标准，自2021年7月1日起实施，把学习者的中文水平分为三等九级，以中文听、说、读、写、译作为五项语言技能。

2022年11月两项国家语委规范：《中小学生普通话水平测试等级标准及测试大纲（试行）》将中小学生普通话水平划分为六级；《汉字部首表》在原有201个主部首基础上增补微调个别附形部首。

中国语言资源保护工程四项国家标准（2025年8月发布）：包括《中国语言资源保护调查规范》系列（汉语方言、少数民族语言、方言文化三项标准）和《中国语言资源保护工作规范》，是指导中国大规模语言资源保护工作的基础性、通用性标准。

人工智能领域两项语言文字规范（2026年3月，教育部、国家语委发布）：《机器合成普通话水平测评等级标准及测评大纲》适用于机器合成语音的普通话水平测评；《人工智能 语料库 基础术语》界定了语料库在属性、建设和应用方面的常用术语和定义。

少数民族语言文字规范标准建设

  • 研制蒙、藏、维、哈、彝等少数民族人名汉字音译转写规范
  • 研制《中国少数民族文字字符总集》，收集了38种古今非拉丁字母少数民族文字，是国内外收录文种最多、字符最全的字符集
  • 开展现代蒙古语常用词、文拉丁转写、现代维吾尔文学语言正字正音、朝鲜语新词术语、彝文规范方案完善、壮文基本词汇等规范标准研制
  • 研制蒙文、藏文拉丁转写国际标准
  • 先后研制发布了200多项国家通用语言文字和少数民族语言文字等的规范标准

（二）积极探索语言文字标准化工作机制

1. 建立健全工作机构

先后成立了负责制定、审定、评测的专业机构：

  • 国家语委语言文字规范标准审定委员会
  • 全国语言文字标准化技术委员会（教育部语用所）
  • 中国语言文字规范标准研究中心（北京语言大学）
  • 中国文字整理与规范研究中心（北京师范大学）
  • 中国文字字体设计与研究中心（北大方正）
  • 国家语委语言文字规范标准测查认证中心（语用所）

2. 探索标准化建设新思路

在规范标准制定中"开门做工作、开门办规范"，采取多种方式广泛听取各界意见和建议，保证规范标准的科学性。以《通用规范汉字表》为例：先后修改90多稿，召开各类会议120多次，收到公开征求意见3141条，海内外专家学者4000多人次参与，涉及近40个主要用字单位和行业主管部门。

3. 探索发布"软性"规范

以《中国语言生活绿皮书》形式发布"软性"规范，满足社会语言生活的多种需求，保证规范标准的科学性和时效性。如《现代汉语常用词表（草案）》《义务教育常用词表（草案）》《日本汉字的汉语读音规范（草案）》等。

四、面临的形势、发展目标和工作任务

（一）面临的形势

语言文字规范标准是语言文字法律法规、方针政策的集中体现，语言文字标准化在语言文字工作中处于重要的基础性地位。面临的主要需求包括：

  • 全面建成社会主义现代化强国
  • 铸牢中华民族共同体意识
  • 建设社会主义文化强国
  • 国家信息化发展
  • 国家治理体系和治理能力现代化

2020年10月全国语言文字会议提出：不断完善语言文字规范体系和标准体系，牢牢掌握中国语言文字国际标准主导权，建立国际中文教育相关标准体系，做好规范标准的发布实施、推广宣传、咨询服务和评测认证工作。

《教育强国建设规划纲要（2024-2035）》明确：健全语言文字规范标准体系，加强网络空间语言文字规范引导，开展语言国情国力调查，深入实施国家语言文化传承发展系列工程。

（二）发展目标

健全语言文字规范标准体系，适应新时代发展需要。规范标准制修订工作取得显著成效，规范标准体系趋于完善；宣传服务、对外传播和体制机制建设效果大幅提升；规范标准效用切实彰显、社会知晓度显著提高；各领域语言文字规范标准应用需求得到进一步满足。

（三）工作原则

坚持统筹兼顾 坚持稳定开放 坚持分类指导 坚持突出实效 坚持有序推进 坚持上下结合

（四）工作任务

  1. 建立和完善语言文字标准化工作长效机制
  2. 加快语言文字规范标准的研制发布
    • 完善国家通用语言文字规范标准
    • 丰富国家通用语言文字工作规范标准
    • 支持面向信息技术及人工智能领域的语言文字规范标准
    • 制定社会应用和信息化急需的少数民族语言文字规范标准
    • 研制服务特殊人群的语言文字规范标准
    • 研制公共服务领域外文译写规范标准和中文国际教育中的语言文字规范标准
    • 主导中国语言文字国际标准的制定
  3. 加强语言文字规范标准的贯彻落实
    • 与相关领域主管部门联动
    • 激发全社会参与积极性，树立"大语言文字"理念
    • 探索宣传培训工作的有效形式
    • 建设相关数据库、资源库及学习平台
    • 开展规范标准符合性测查认证
  4. 重视语言文字标准化人才队伍建设

需要注意的几个问题

  1. 规范标准制定要有计划性和系统性
  2. 规范标准制定要以科学研究为基础
  3. 规范标准制定要广泛征求社会意见
  4. 规范标准发布后要开展深入细致的宣传工作

五、语言文字规范标准制修订流程

制度依据

《国家语委语言文字规范标准审定委员会章程》（2018年修订）——为贯彻实施《国家通用语言文字法》，保证语言文字规范标准审定工作的规范性和科学性而制定。
《国家语委语言文字规范标准管理办法》（2018年修订）——国家语委为语言文字规范标准主管部门，负责语言文字规范标准管理工作。教育部语信司或由语信司委托的专业机构具体承担语言文字规范标准的立项、研制、审定、实施等的组织协调工作。

制修订流程

  1. 研制：研制单位和研制组对所研制规范标准的质量负责。根据语言文字法律政策的调整、社会发展和语言生活需要，研制单位和研制组形成规范标准送审稿。
  2. 审定：规范标准送审稿原则上采用会议形式审定。通过后，研制单位和研制组根据审定意见形成报批稿，提交国家语委审核发布。
  3. 审批、发布：语言文字规范（GF）由国家语委主任签发，由教育部、国家语委发布。国家标准（GB）经国家语委主任批准后按相关规定报国家标准主管部门审批发布。
  4. 复审：根据语言文字法律政策调整、社会发展和语言生活需要适时进行维护性复审。复审结果：继续有效、修订、废止。
结语：语言文字规范化工作坚持政府主导、学界引领、群众参与的工作格局。
